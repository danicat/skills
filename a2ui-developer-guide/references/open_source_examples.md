# Open Source Examples and Ecosystem Projects
Source: Extracted from a2ui.org community and ecosystem pages & the official [A2UI Samples Registry](https://github.com/a2ui-project/a2ui/tree/main/samples).

Here are several open source projects, official examples, and community implementations of A2UI. Use these as references when asked to build or look for examples of A2UI applications.

## Official Samples (A2UI Repository)
The `a2ui-project/a2ui` GitHub repository contains the main registry of examples:
*   **[Simple Chat](https://github.com/flutter/genui/blob/main/examples/simple_chat/README.md)**: A Dart agent with a Flutter renderer.
*   **[Restaurant Finder](https://github.com/a2ui-project/a2ui/tree/main/samples)**: A Gemini-powered Python agent that helps users book tables. Showcases dynamic forms. Available for 4 renderers:
    *   [Angular](https://github.com/a2ui-project/a2ui/blob/main/samples/client/angular/README.md)
    *   [Flutter](https://github.com/a2ui-project/a2ui/blob/main/samples/client/flutter/restaurant_finder/README.md)
    *   [Lit](https://github.com/a2ui-project/a2ui/blob/main/samples/client/lit/shell/README.md)
    *   [React](https://github.com/a2ui-project/a2ui/blob/main/samples/client/react/shell/README.md)
*   **[Verdure](https://github.com/flutter/genui/blob/main/examples/verdure/README.md)**: A Python agent with a Flutter renderer.
*   **[Component Gallery](https://github.com/a2ui-project/a2ui/tree/main/samples/client/angular)**: An interactive showcase of all basic A2UI components built in Angular.
*   **[Rizzcharts](https://github.com/a2ui-project/a2ui/tree/main/samples/community/agent/adk/rizzcharts/python)**: Demonstrates how to build and integrate custom components (specifically charts) using the ADK.
*   **[Orchestrator](https://github.com/a2ui-project/a2ui/tree/main/samples/community/agent/adk/orchestrator)**: Shows how to pass A2UI messages from remote subagents.

## Dev Tools
*   **[Catalog Gallery](https://github.com/flutter/genui/blob/main/dev_tools/catalog_gallery/README.md)**: A Flutter application for previewing catalogs.
*   **[Composer](https://github.com/flutter/genui/blob/main/dev_tools/composer/README.md)**: A Flutter dev tool.

## Community Renderers & Libraries
*   **[json-render](https://json-render.dev/docs/a2ui)**: Vercel's React library for rendering A2UI component catalogs via Zod schemas.
*   **[A2UI-Android](https://github.com/lmee/A2UI-Android)**: A community-built Jetpack Compose renderer containing over 20 components.
*   **[a2ui-react-native](https://github.com/sivamrudram-eng/a2ui-react-native)**: A React Native renderer for iOS and Android.
*   **[Lynx A2UI](https://lynxjs.org/next/react/genui/a2ui.html)**: A ReactLynx renderer for A2UI (v0.9).

## Products Using A2UI
*   **[OpenClaw Canvas](https://docs.openclaw.ai/platforms/mac/canvas)**: Uses A2UI to render agent-generated UI on connected devices via its canvas system.
*   **[CopilotKit / AG-UI](https://www.copilotkit.ai/)**: Provides full-stack agentic frameworks with day-zero A2UI compatibility for React, Vue, and Angular apps.
